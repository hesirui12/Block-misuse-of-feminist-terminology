# 「网络环境净化助手」插件介绍

## 🌈 产品初衷
我们观察到在性别议题的讨论中，有时会出现词汇滥用影响理性沟通的情况。为此开发了这款「对话优化插件」，帮助用户过滤偏激化用语，回归平等对话本质，让网络空间更聚焦建设性讨论。

## ✨ 核心功能亮点

### 🎯 精准语义识别
- 智能识别超过200个敏感词的 **滥用场景**
- 采用NLP技术区分合理讨论与攻击性言论
  ```示例
  ✔️ 保留："我们需要推动性别平等教育"
  ✖️ 过滤："所有男性都是..."
  ```

### 🎨 个性化定制
| 功能       | 说明                          |
|------------|-----------------------------|
| 词库DIY    | 支持导入/导出个性化词库        |
| 语境白名单 | 添加信任站点（如学术论坛）      |
| 替换方案   | 可选[***]/[表情]/自定义替换文字 |

### 🔒 隐私守护
- 所有处理均在本地完成
- 绝不收集用户浏览数据
- 开源处理逻辑可审查

## 📍 适用场景
- 社交媒体浏览
- 新闻评论区管理
- 亲子网络防护
- 公共空间设备

## 🛠️ 使用指南
1. 从插件市场一键安装
2. 开启智能过滤模式（默认）
3. 通过「咖啡杯」图标快速调整设置

## 🤝 我们主张
本工具旨在：
- 消除网络戾气，回归理性对话
- 保护未成年人免受极端言论影响
- 帮助不同立场者找到对话可能
